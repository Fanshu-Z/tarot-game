# 塔罗牌抽卡网页游戏 - 使用说明

## 🎮 快速开始

### 方法一：本地直接运行（推荐）
1. 将 `tarot_game_files` 文件夹复制到任意位置
2. 双击打开 `tarot_game.html` 文件
3. 游戏即可在浏览器中运行

### 方法二：部署到免费平台（便于分享）

#### 选项A：GitHub Pages（推荐，完全免费）
1. **注册 GitHub 账号**：访问 https://github.com
2. **创建新仓库**：
   - 点击右上角 "+" → "New repository"
   - 仓库名：例如 `tarot-game`
   - 设为 Public（公开）
   - 点击 "Create repository"
3. **上传文件**：
   - 点击 "uploading an existing file"
   - 将整个 `tarot_game_files` 文件夹内的所有文件拖入
   - 点击 "Commit changes"
4. **启用 GitHub Pages**：
   - 进入仓库设置：Settings → Pages
   - 在 "Source" 下选择 "main" 分支
   - 点击 Save
5. **获取游戏链接**：
   - 几分钟后访问 `https://你的用户名.github.io/tarot-game/`
   - 把这个链接分享给朋友即可

#### 选项B：Vercel（简单，支持自定义域名）
1. 访问 https://vercel.com 注册账号
2. 点击 "New Project"
3. 导入 GitHub 仓库或直接拖入文件
4. 点击 "Deploy"
5. 获得一个 `https://你的项目名.vercel.app` 的链接

#### 选项C：Netlify（最简单，拖拽即用）
1. 访问 https://app.netlify.com/drop
2. 直接将 `tarot_game_files` 文件夹拖入页面
3. 等待上传完成，获得游戏链接

## 📱 移动端分享

### 微信分享
1. 使用 GitHub Pages 或 Vercel 部署后，获得 HTTPS 链接
2. 直接在微信中发送链接即可
3. 由于游戏已适配响应式设计，手机上体验优秀

### 其他社交平台
- 将部署后的链接复制到任何社交平台即可分享
- 支持：微博、QQ、Telegram、WhatsApp 等

## 🎨 自定义说明

如需修改游戏内容：
- 用文本编辑器（如记事本、VS Code）打开 `tarot_game.html`
- 修改卡片解读、颜色、文字等
- 保存后刷新页面即可看到效果

## 💡 使用技巧

1. **离线使用**：整个游戏完全本地化，无需联网（除了首次加载）
2. **移动端优化**：已适配手机、平板、电脑各种屏幕尺寸
3. **隐私安全**：所有数据在本地处理，不上传任何信息

## 📞 技术支持

如遇到问题：
- 确保所有图片文件与 HTML 文件在同一目录
- 使用现代浏览器（Chrome、Firefox、Safari、Edge）
- 清除浏览器缓存后重试
